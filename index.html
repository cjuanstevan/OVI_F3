OBJETIVO GENERAL

El estudiante al final del curso estará en la capacidad de diseñar e implementar apropiadamente los pasos en un modelo relacional sobre los sistemas de gestión de bases de datos como (Oracle, MYSQL, SQL Server o Teradata) y utilizar de manera práctica los conceptos básicos en bases de datos relacionales y comprender los conceptos relacionados con el uso de LDD, LMD, LCD.

OBJETIVOS ESPECIFICOS
  • Conocer y utilizar los conceptos básicos sobre los sistemas de gestión de bases de datos relacional
  • Aprender a crear una base de datos con un modelo relacional sobre los SGBD.
  • Realizar de manera práctica procedimientos almacenados, aplicando LDD, LMD, LCD
