1. Conocer y utilizar los conceptos básicos sobre los sistemas de gestión de bases de datos relacional.

En esta sección el estudiante aprenderá a crear una base de datos relacional, aplicando los conceptos básicos para determinar que campos son necesarios al momento de crear una tabla y sus respectivas relaciones. Los temas que trataremos son:


  • Crear una base de datos
  • Crear las tablas de la base de datos



2. Aprender a crear una base de datos con un modelo relacional sobre los SGBD.

En esta sección el estudiante obtendrá los conocimientos básicos para crear o diseñar una base de datos relacional sobre un SGBD. Las temáticas son:


  • Modelo entidad-relación.
  • Notación o representación gráfica del modelo ER
  • Diseño práctico de una base de datos relacional



3. Realizar de manera práctica procedimientos almacenados, aplicando LDD, LMD, LCD.

En esta sección el aplicaremos los conocimientos aprendidos realizando la siguiente actividad. Los temas que trataremos son:


  • Crear Índices
  • Crear vistas
  • Modificación de Tablas
  • Borrados de Bases de Datos, tablas
  • Uso de LDD, LMD, LCD.
  • Realizar de manera práctica procedimientos almacenados
